## 프로젝트 소개
대도시에는 지하철을 이용하는 수많은 사람이 있습니다. 하지만 지하철 도착 정보를 얻기 위해서는 지도 앱을 직접 켜고 복잡한 메뉴를 통해 여러 화면을 열어야 하는 번거로움이 있습니다. '탈래말래'는 이 문제를 해결하기 위해 사용자의 GPS를 기반으로 현 위치에서 탑승 가능한 지하철 도착 정보를 모바일 푸시 형식으로 제공하는 앱입니다.
